# iOS App Binary

This folder is where the **iOS .app bundle under test** lives (simulator only).

For real devices you'd use a `.ipa` with proper signing — out of scope for this demo framework.

## Download

```bash
curl -L -o iOS.Simulator.SauceLabs.Mobile.Sample.app.2.7.1.zip \
  https://github.com/saucelabs/sample-app-mobile/releases/download/2.7.1/iOS.Simulator.SauceLabs.Mobile.Sample.app.2.7.1.zip
unzip iOS.Simulator.SauceLabs.Mobile.Sample.app.2.7.1.zip
mv "SauceLabs-Demo-App.app" .
```

Or grab the latest release from:
👉 https://github.com/saucelabs/sample-app-mobile/releases
